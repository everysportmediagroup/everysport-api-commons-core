everysport-api-commons-core
===========================

## Maven
See http://search.maven.org/ for latest version

```
<dependency>
    <groupId>com.everysport</groupId>
    <artifactId>everysport-api-commons-core</artifactId>
    <version>${es.commons.version}</version>
</dependency>
```

## Build and install
```
mvn clean install
```
